# Contenu de ce répertoire

- Le fichier `CMakeLists.txt` a été écrit par Michel Simatic.
- Les fichiers `sqlite3.c`, `sqlite3.h` et `sqlite3ext.h` ont été obtenus en téléchargeant la version "*amalgation*" de *sqlite à partir de [la page de téléchargement](https://www.sqlite.org/download.html). Pour être plus précis, ces trois fichiers ont été extraits de [sqlite-amalgamation-3500100.zip](https://www.sqlite.org/2025/sqlite-amalgamation-3500100.zip), donc la version 3.50.01 du 2025-06-06 de [*sqlite*](https://www.sqlite.org).
